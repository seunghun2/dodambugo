## 🚀 프로젝트 개요

**마음부고**는 온라인 모바일 부고장 서비스입니다.

### 기술 스택
- **Frontend**: Next.js 15, React, TypeScript
- **Styling**: CSS (커스텀 디자인 시스템)
- **Database**: Supabase
- **Deployment**: Vercel

### 주요 페이지
- `/` - 메인 페이지
- `/create` - 템플릿 선택
- `/create/[templateId]` - 부고장 작성
- `/create/complete/[id]` - 작성 완료
- `/view/[id]` - 부고장 조회
- `/search` - 부고 검색
- `/faq` - 자주 묻는 질문

## ⚠️ 리스크 및 고려사항

### 어려운 점
1. **마진이 얇음** - 광고비 빼면 많이 안 남음
2. **스케일 한계** - 시장 크기 제한 (니치 마켓)
3. **운영 부담** - 장례는 급해서 CS 많음
4. **24시간** - 부고는 새벽에도 옴
5. **광고비 증가 한계** - 키워드 클릭 수 한정

### 스케일업 방법
1. **SEO/블로그** - 시간 걸리지만 무료
2. **장례식장 제휴** - B2B 가능성
3. **멀티 사이트** - 광고 슬롯 점유율 ↑
4. **전환율 최적화** - UX 개선

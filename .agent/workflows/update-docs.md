## 작업 완료 후 반드시 수행

모든 기능 구현/수정 작업이 끝나면:

1. **CHANGELOG.md 업데이트**
   - 날짜, 변경 내용 기록
   - 파일 경로: `/docs/CHANGELOG.md`

2. **관련 문서 업데이트**
   - 디자인 변경 → `DESIGN_SYSTEM.md`
   - 새 기능 → `FEATURES.md`
   - API 변경 → `API.md`
   - 할 일 변경 → `TODO.md`

3. **Git 커밋**
   - 코드 + 문서 함께 커밋
   - 메시지에 변경 내용 명시

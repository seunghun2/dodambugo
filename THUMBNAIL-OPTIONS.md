# 🎨 부고장 썸네일 개선 방안

## 현재 상황
- 4가지 템플릿 이미지 위에 텍스트 오버레이
- 이미지와 텍스트가 중복되어 복잡함
- 가독성 문제 발생 가능

---

## 💡 개선 방안 (3가지 옵션)

### 옵션 1: 썸네일만 표시 (가장 심플) ⭐ 추천
**장점:**
- 가장 깔끔하고 토스 디자인 철학에 부합
- 이미지 자체가 아름다움
- 텍스트 중복 없음

**구현:**
```html
<!-- 헤더에서 텍스트 오버레이 완전 제거 -->
<div class="bugo-header">
    <div class="header-template-bg"></div>
    <!-- 텍스트 오버레이 삭제 -->
</div>

<!-- 고인 정보 섹션에서 애도 문구 표시 -->
<section class="condolence-section">
    <p class="condolence-main">故 홍길동 님께서 11월 25일 별세하셨기에</p>
    <p class="condolence-sub">삼가 알려드립니다.</p>
</section>
```

---

### 옵션 2: 반투명 박스로 가독성 개선
**장점:**
- 텍스트가 명확하게 보임
- 이미지도 함께 활용
- 세련된 느낌

**구현:**
```css
.condolence-message-header {
    background: rgba(255, 255, 255, 0.9);
    backdrop-filter: blur(8px);
    padding: 16px;
    border-radius: 12px;
    box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
}

.template-flower .condolence-message-header {
    background: rgba(0, 0, 0, 0.7);
}
```

---

### 옵션 3: 썸네일 크기 축소 + 상단 배치
**장점:**
- 전체 레이아웃이 깔끔
- 스크롤 적음
- 정보 전달이 빠름

**구현:**
```css
.bugo-header {
    max-width: 280px; /* 380px → 280px */
    padding-bottom: 140%; /* 165% → 140% */
    margin: 12px auto;
}

.condolence-message-header p {
    font-size: 9px;
    line-height: 16px;
}
```

## 🎯 가장 추천하는 방안

### **옵션 1: 썸네일만 표시 (텍스트 오버레이 제거)**

#### 이유:
1. ✅ **토스 디자인 철학**: Simple & Clear
2. ✅ **이미지 품질**: 템플릿 자체가 이미 완성도 높음
3. ✅ **가독성**: 텍스트 중복 없음
4. ✅ **모바일 최적화**: 작은 화면에서도 깔끔

#### 레이아웃 구조:
```
┌─────────────────────┐
│   템플릿 썸네일      │  ← 이미지만
│   (訃告/謹弔 등)     │
└─────────────────────┘

┌─────────────────────┐
│ 故 홍길동 님께서     │  ← 애도 문구 섹션 (새로 추가)
│ 11월 25일 별세...   │
└─────────────────────┘

┌─────────────────────┐
│ 고인 정보           │
│ - 고인: 故 홍길동   │
│ - 향년: 75세        │
└─────────────────────┘
```

## 📋 추가 제안

### 다크모드에서의 썸네일
- 다크모드에서는 썸네일에 약간의 그림자 추가
- 흰색 배경 템플릿(basic, ribbon, border)은 테두리 추가

```css
.dark .bugo-header {
    box-shadow: 0 4px 16px rgba(0, 0, 0, 0.3);
    border: 1px solid #3A3A3C;
}

.dark .template-flower .bugo-header {
    border: none; /* 검정 배경은 테두리 불필요 */
}
```
